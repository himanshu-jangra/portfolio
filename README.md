# Himanshu Jangra - Work In Progress Portfolio

A minimalist, **Neo-Brutalist** style landing page designed to serve as a "Work In Progress" placeholder or a social link hub. The design features high-contrast borders, hard drop shadows, and responsive grid layouts.

## 🎨 Design Features

- **Neo-Brutalism Aesthetic:** Uses thick borders (`2.5px`), monochromatic colors, and hard shadows.
- **Interactive Buttons:** Buttons feature a tactile "press-in" animation where the shadow disappears and the button moves flush with the background on hover.
- **Responsive Layout:**
  - **Desktop:** 6-column social grid.
  - **Tablet:** 3-column grid.
  - **Mobile:** 2-column grid (stacked vertically on very small screens).
- **Typography:** Uses **Mona Sans** (Condensed) for a bold, modern look and **Gurmukhi MN** for localized footer text.
- **Zero Dependencies:** Pure HTML, CSS, and Vanilla JavaScript. No external frameworks or icon libraries required.

## 📝 Fonts used
- **English:** Mona Sans Condensed
- **Punjabi:** Gurmukhi MN
